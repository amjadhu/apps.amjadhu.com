# apps.amjadhu.com

A visually appealing gallery of apps and tools built by Amjad Hussain.

**Live:** [apps.amjadhu.com](https://apps.amjadhu.com)

## Apps

- **[The Commission](https://amjadhu.github.io/the-commission/)** - Seahawks fan hub with news feed, hot takes, and reactions
- **[AI Terminal](https://ai-terminal-one.vercel.app)** - Bloomberg terminal-style dashboard
- **[College Explorer](https://amjadhu.github.io/college-explorer/)** - Interactive college comparison tool
- **[Vantage](https://github.com/amjadhu/vantage)** - AI-powered intelligence dashboard (Claude + Next.js)
- **[Syslens](https://github.com/amjadhu/syslens)** - Cross-platform CLI for system information

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript (no frameworks or dependencies)
- Deployed to [GitHub Pages](https://pages.github.com/) via GitHub Actions
- Custom domain: [apps.amjadhu.com](https://apps.amjadhu.com)

## Local Development

Open `index.html` in a browser. No build step required.

## Adding a New App

Add a card element inside the `.grid` div in `index.html` and push to `main`.

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via `.github/workflows/deploy.yml`.
